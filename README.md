## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/MatPizzolo/sabujcha-landing.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run app
   ```sh
   npm run start
   ```
